## 系統架構說明

```
每個專案 = 一張專案卡
專案卡包含：背景 → 目標 → KPI → 里程碑 → 任務 → 回顧
看板狀態：📋 規劃中 → 🚀 進行中 → ⏸ 暫停 → ✅ 完成
```
